## Skills

- **typography** — 字体分类、文本间距、字形解剖、CSS 单位、渲染工程
- **color-system** — 色彩感知、色彩空间、调色盘架构、Design Tokens、无障碍 WCAG
- **layout-typesetting** — 布局基础、栅格系统、布局模式、响应式策略、CSS 工程
